updateHosts
============

##运行环境
- 操作系统：Windows
- python版本：2.x

##功能描述
- 从imouto.host下载hosts文件并更新本地hosts
- 可添加启动项
- 生成错误日志errorLog.txt
- 可去除AdBlock部分，默认启用

##使用方法
下载**updateHosts.py**和**addToStarup.js**到本地，直接运行updateHosts.py更新hosts，点击addToStarup.js讲py文件的快捷方式加入启动项
